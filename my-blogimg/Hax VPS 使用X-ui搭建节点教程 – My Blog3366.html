Hax VPS 使用X-ui搭建节点教程

7月 1, 2022 admin

Hax这个网站提供了免费的IPV6 VPS，以及免费的网络虚拟空间的服务。

准备材料:

一枚TG账号

注册VPS

1.打开官网，点击右上角的“Register”注册

2.在Telegram上打开@HaxTG_bot，输入/getid命令获取自己的Telegram ID，复制粘贴到网页中，点击“Submit”按钮

3.稍后会在Telegram上收到bot发来的验证码，输入验证码，设置密码，登录账号

4.设置好VPS参数之后，点击“Create VPS”按钮

5.等待1-3分钟，然后就出现了VPS的登录信息

连接VPS

1.打开test-ipv6.com，检查自己的IPV6连接情况

如有IPV6

2.复制粘贴VPS的IPV6地址

3.打开Xshell，新建一个会话

按照下图设置

4.点击“用户身份验证”，用户名输入“root”，密码输入你设置的密码

5.点击“连接”

如没有IPV6

用官网Web Terminal直接连接ssh

1.打开ssh.hax.co.id 

2.输入自己VPS的IPV6地址，用户名和密码，点击“Connect”

用ssh软件登录方法：

需要IPV4转发才可以登录ssh

1.打开Hax的IPV4 to IPV6页面

2.按照下图设置，点击“Request to forward port”

1.打开Xshell，新建一个会话

2.按照下图设置

5.点击“用户身份验证”，用户名输入“root”，密码输入你设置的密码

6.点击“连接”

安装x-ui面板搭建节点方法

使用你获得的IPV4打开SSH，或者使用上面官方的SSH连接.

在VPS内安装WARP：

1.首先打开TUN模块（如果是KVM机器可跳过此步骤）。以Hax官网为例，打开官网的控制面板，点击“Enable TUN”按钮，稍等片刻后再点击“Restart”按钮重启VPS

vps开启tun
2.我们连接SSH，输入cat /dev/net/tun命令，检查TUN模块状态，如出现“cat: /dev/net/tun: File descriptor in bad state”时即为开启成功

3.复制以下安装命令

wget -N https://cdn.jsdelivr.net/gh/fscarmen/warp/menu.sh && bash menu.sh

4.选择语言，输入2选择简体中文

5.输入“1”安装WARP（IPV4连接走WARP的流量），当然你输入“2”也可以（全部连接走WARP的流量）

VPS使用x-ui搭建节点

1.安装x-ui

复制粘贴以下代码，并运行

bash <(curl -Ls https://raw.githubusercontent.com/vaxilu/x-ui/master/install.sh)

注意：x-ui面板安装到最后时会提示面板的端口，用户名，密码，是否修改，选择y 是修改，选择n选择默认，默认的端口是54321，用户名 admin 密码 admin

x-ui面板安装完成.

1.再添加一条IPV4转发    无论你是否修改过端口都要添加一条IPV4转发

2.1.打开Hax的IPV4 to IPV6页面

3.按照下图设置，下图是没有修改端口的设置方法：点击“Request to forward port”

登录面板：

1.没有修改过端口的使用“IPv4:54321”（[IPV6]:54321）登录x-ui面板，默认端口是54321，默认的用户名和密码都是“admin”

http://91.134.238.134:54321/

[2a01:4f8:16a:2695:face:19a5:ba0b:0001]:54321

2.如果你在安装x-ui面板时选择了y ，比喻你修改的端口是8888，把你修改的端口记下来，等面板安装好之后去 Hax的IPV4 to IPV6页面添加一条端口8888转发

如果修改过端口的.使用“IPv4:54321”（[IPV6]:8888）登录x-ui面板，端口是8888，用户名和密码是你自己自定义设置的.

http://91.134.238.134:8888/

[2a01:4f8:16a:2695:face:19a5:ba0b:0001]:8888

1.需要一个域名

2.在CF中DNS中，给域名添加AAAA记录为IPv6地址；

3.在SSL/TLS中，选择“完全”，申请客户端证书；

复制证书里的公匙和密匙的内容，把复制好的公匙和密匙粘贴在入站列表创建一个节点的对应公匙和密匙内容处。

4.入站列表创建一个节点，X-ui按下图所示配置

hax vps续期方法：

https://hax.co.id/vps-renew/

在“Extend VPS”处，输入官网地址hax.co.id，输入数字验证码和完成谷歌验证码，点击“Extend VPS”即可
